@thebespokepixel/badges

Usage

Installation

npm install --save @thebespokepixel/badges

Setup

Require (or import) the module…

const badges = require('@thebespokepixel/badges')

…then do something with it.

function renderBadges() {
    badges('readme').then(markdown => {
        /* Output markdown to console... */
        console.log(markdown)
    })
}

API

render → Promise

src/index.js

Render project badge configuration as markdown.

render(context: String, asAST: [Boolean](default false)) → Promise
Parameters
context (String) The desired render context i.e: readme , docs as defined in package.json .
asAST ([Boolean](default false)) Render badges as MDAST
Returns
Promise: A promise that resolves to the markdown formatted output.
